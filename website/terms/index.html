2. Description of Service

SpendGuard is a real-time authorization API for AI agent financial actions. The Service evaluates planned financial actions (refunds, credits, discounts, vendor payments) against user-defined policies and returns one of three decisions: allow, block, or escalate.

3. Nature of Service and Accuracy Disclaimer

SpendGuard is an automated authorization service that evaluates financial actions against user-defined policies. The Service employs deterministic, rule-based logic as its primary decision engine, supplemented by an AI-assisted semantic classification layer for ambiguous inputs.

While SpendGuard is engineered with multiple safeguards — including deterministic rule evaluation, duplicate detection, and immutable audit logging — no automated system, including those incorporating artificial intelligence components, can guarantee 100% accuracy in all circumstances.

The AI-assisted semantic classification layer is designed to resolve ambiguous action types only and does not override rule-based decisions. However, as with any system involving automated classification, edge cases may arise where inputs are interpreted differently than intended.

SpendGuard is a decision-support and policy-enforcement tool. It is not a substitute for human judgment on high-value, high-risk, or unusual financial transactions. Customers are responsible for:

  • Defining appropriate policies and rules for their use case
  • Testing policies thoroughly using the simulation endpoint before production deployment
  • Implementing appropriate human review processes for transactions that warrant additional oversight
  • Monitoring audit logs and adjusting policies as needed

SpendGuard is provided to assist and augment your financial controls, not to replace them entirely.
